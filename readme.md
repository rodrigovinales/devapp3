## _Clase 14 Desarrollo de Aplicaciones_

## Detalle

Se solucionan y se agregan detalles como:
_ Alert en carrito para confirmar la compra o seguir comprando.
_ Vaciado de carrito al confirmar la compra. (Se agrega un reducer para tal accion).
_ Validacion de carrito vacio, en caso de estarlo, vuelve a Home para seleccion de productos.
